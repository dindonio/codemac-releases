## Features

### Editor
- **Custom Core Text rendering engine** — instant loading of large files, 60fps scrolling
- **Syntax highlighting** for 31 languages (10 with tree-sitter AST parsing)
- **Multi-cursor editing** — Cmd+D to select next, Option+Click to add cursor
- **Code folding** — collapse functions, classes, blocks
- **Rainbow brackets** — color-coded nesting depth
- **Indent guides** — vertical lines at each indentation level
- **Sticky scroll** — parent scope headers stick to top
- **Find & Replace** with regex, Find in Files
- **Minimap** — scrollable, syntax-colored overview
- **Bookmarks** — double-click gutter to mark lines
- **Split view** — vertical and horizontal editor splitting
- **Snippet system** — 50+ built-in, Tab to expand
- **Word wrap**, auto-indent, auto-close brackets

### AI Integration
- **AI Create Code** — describe what you need, code is generated and inserted at cursor
- **AI Copilot Inline** — ghost text suggestions while typing
- **AI Actions** — Explain, Refactor, Fix, Document, Write Tests
- **Providers** — Claude, OpenAI GPT-4, Google Gemini (bring your own API key)
- **Secure** — API keys stored in macOS Keychain, consent dialog before first use

### Git Integration
- **Git panel** in sidebar — status, stage/unstage, commit
- **Git gutter** — green/blue/red indicators per line
- **Git blame inline** — author, date, commit summary
- **Push / Pull / Clone** with keyboard shortcuts

### Language Server Protocol (LSP)
- **Auto-completion**, signature help, hover info
- **Go to definition**, code actions, rename symbol
- **Inline diagnostics** + problems panel
- **13 language servers** auto-discovered

### Compare / Diff
- Side-by-side diff, compare with saved/clipboard/HEAD
- Compare two files or open tabs

### Tools
- **6 built-in themes** — One Dark, Midnight, Ocean, Forest, Light, Sunset
- Command Palette, Quick Open, Function List
- Clipboard History, Macro Recording, Column Editor
- Line Operations, Case Conversion, Hashing, Encoding
- 30+ file types supported, Open With from Finder
